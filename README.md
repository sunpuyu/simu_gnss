# simu_gnss

室内天线转发卫星信号定位系统仿真工具。基于 RINEX 3.04 星历文件，通过加权最小二乘法（WLS）仿真室内双天线 GNSS 信号转发场景下的定位结果。

## 功能概述

- 解析 RINEX 3.04 NAV 星历文件（支持 GPS / BDS）
- 计算指定时刻、指定参考点的可见卫星及其方位角 / 仰角，生成星空图
- 配置室内地板尺寸与天线位置，可视化穿刺点分布
- **单天线仿真**：每个穿刺点选取最近天线，等权加权最小二乘解算（4 参数）
- **双天线仿真**：GPS/BDS 双系统，基于 SNR 指数定权模型，加权最小二乘解算（5 参数，含系统间偏差 ISB）
- 仿真结果导出为 CSV 文件

## 界面预览

| 左面板 | 右面板 |
|--------|--------|
| 星历解析 + 星空可视化 | 室内建模 + 仿真结果 |

- **左面板**：读取星历文件 → 设置基准点坐标和参考时间 → 确认计算 → 查看星空图（蓝色 = GPS，红色 = BDS）
- **右面板**：配置地板尺寸 → 配置天线位置 → 天线选星绑定 → 执行仿真 → 导出 CSV

## 快速开始

本项目为纯前端静态页面，无需安装任何依赖或构建步骤。

```bash
# 克隆仓库
git clone https://github.com/<your-username>/simu_gnss.git
cd simu_gnss
```

用任意支持 ES Modules 的静态服务器启动（需要 HTTP 协议，不能直接双击 `index.html`）：

```bash
# 方式一：Python
python3 -m http.server 8080

# 方式二：Node.js npx
npx serve .

# 方式三：VS Code Live Server 插件
# 右键 index.html → Open with Live Server
```

打开浏览器访问 `http://localhost:8080`。

`test/` 目录内提供了一份示例星历文件可直接使用：

```
test/BRDC00IGS_R_20260010000_01D_MN.rnx
```

## 使用流程

### 1. 加载星历文件

点击「读取星历文件」，选择 RINEX 3.04 NAV 文件（`.rnx` / `.nav`）。加载成功后自动显示星历时间范围。

### 2. 计算可见卫星

填写基准点的经度、纬度、高程（WGS84），以及参考时间（格式 `YYYY-MM-DD HH:MM:SS`），点击「确认计算」。星空图将实时更新，鼠标悬停卫星节点可查看卫星 ID 和仰角。

### 3. 配置室内场景

- 点击右面板**地板区**，在弹窗中输入地板长度和宽度（米），确认后地板网格自动按 1 m 间距生成穿刺点。
- 点击右面板**天线配置区**中的天线标记，分别设置两根天线的经度、纬度、高程和离地高度。

### 4. 天线选星绑定

点击「天线选星」，在弹窗中将可见卫星分配给两根天线（双天线仿真时一根绑定 GPS，另一根绑定 BDS）。

### 5. 执行仿真

- **单天线仿真**：每个穿刺点自动选取距离最近的天线，等权最小二乘解算 `[X, Y, Z, cδt]` 4 个参数。卫星观测值 ≥ 4 时参与解算。
- **双天线仿真**：每个穿刺点同时接收两根天线信号，采用 SNR 定权模型（`SNR = 45 − d`，`w = 10^(SNR/10)`，SNR < 20 时权重为 0），解算 `[X, Y, Z, cδt_GPS, ISB_BDS]` 5 个参数。卫星观测值 ≥ 5 时参与解算。

### 6. 导出结果

仿真完成后，点击弹窗中的「导出 CSV」按钮，下载包含仿真点经纬度坐标的结果文件。

## 算法说明

### 观测值构造

卫星观测值为**折线距离**：

```
ρ = |卫星 → 天线| + |天线 → 穿刺点|
```

其中各坐标均为 WGS84 ECEF（地心地固坐标系）下的已知值。

### 加权最小二乘

线性化后的观测方程：

```
H · δx = l
```

法方程：

```
(HᵀWH) δx = HᵀWl
```

采用高斯消去法求解（部分主元选取），全程使用双精度浮点保证精度。

**设计矩阵 H 构成：**

| 列 | 说明 |
|----|------|
| 1–3 | 方向余弦 `(X₀ − Xˢ) / ρ₀` 等 |
| 4 | 接收机钟差（GPS 系统，值为 `1`） |
| 5（仅双天线） | BDS 系统间偏差 ISB（BDS 卫星取 `1`，GPS 卫星取 `0`） |

### 坐标转换

- **LLA → ECEF**：WGS84 椭球参数（`a = 6378137 m`，`f = 1/298.257223563`）
- **ECEF → LLA**：迭代法（Bowring 方法）
- **ECEF → ENU**：仰角 / 方位角计算用于星空图渲染

## 技术栈

- **纯前端**：HTML5 + CSS3 + Vanilla JavaScript（ES6 Modules，无框架，无构建工具）
- **可视化**：HTML5 Canvas 2D（星空图 + 地板平面图）
- **数据格式**：RINEX 3.04 NAV（GNSS 卫星星历）
- **坐标系**：WGS84（LLA / ECEF / ENU）
- **算法**：加权最小二乘法 + 高斯消去法

## 浏览器兼容性

支持所有现代浏览器（Chrome 80+、Firefox 75+、Edge 80+、Safari 14+）。需要启用 ES Modules 支持（即不支持 IE）。
